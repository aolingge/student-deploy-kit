# Security Policy

## Supported Content

This repository provides templates and scripts. Security fixes for templates, examples, and documentation are welcome.

## Reporting A Security Problem

Open a private security advisory on GitHub if available, or create an issue without including secrets.

Do not paste:

- private keys
- server passwords
- API tokens
- database connection strings
- real `.env` files

## Safe Usage

Always review scripts before running them with `sudo`.
