# Research Notes

Date: 2026-04-25

## Why This Project

The goal is to build a GitHub repository with stronger star and follow potential than a generic portfolio template.

## Compared Directions

### Student Deploy Kit

Good fit because the search space is fragmented. Many repositories solve only one narrow case, such as one Spring Boot + Nginx demo or one Django deployment script. A beginner-friendly kit can combine the repeated pain points into one memorable repository.

### MCP Server Template

The MCP topic is hot in 2026, but templates are already appearing quickly and the maintenance bar is higher. A weak template will be lost in a crowded ecosystem.

### Portfolio Template

Useful, but heavily saturated. Most users fork and customize instead of contributing back.

## Positioning

Student Deploy Kit targets students and indie developers who need to deploy real projects on a VPS without learning the entire DevOps stack first.

## Growth Hooks

- Search keywords: Spring Boot deploy, Nginx config, Vue deploy, React deploy, VPS deploy, Docker Compose, GitHub Actions.
- Contribution hooks: one template per framework or cloud provider.
- Beginner issues: docs fixes, cloud notes, new config examples.
