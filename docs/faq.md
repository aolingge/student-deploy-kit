# FAQ

## Is this for production?

It is a starter kit. Read every template, replace placeholders, and test before production.

## Do I need Docker?

No. The repository includes both classic VPS deployment and Docker examples.

## Why not use Kubernetes?

Most student projects do not need Kubernetes. Start with one VPS, Nginx, systemd, and logs. Add Kubernetes when the project really needs it.

## Can I use this for my graduation project?

Yes. The templates are designed for typical graduation projects: Spring Boot backend, Vue/React frontend, and a small cloud server.

## How do I contribute?

Add a template, improve a guide, or document one real deployment problem and solution.
