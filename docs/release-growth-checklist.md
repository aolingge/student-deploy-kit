# Release Growth Checklist

Use this checklist after publishing the repository.

- Add topics: `springboot`, `nginx`, `deploy`, `docker`, `github-actions`, `student`, `vps`, `frontend`.
- Pin the repository on your GitHub profile.
- Add screenshots or terminal GIFs.
- Publish a short Chinese article with the title: `我做了一个学生项目一键部署工具包`.
- Share one concrete deployment case instead of a generic introduction.
- Create 5 good first issues before promoting.
- Reply to every issue in the first week.
